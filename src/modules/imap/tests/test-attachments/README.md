# Test Attachments

This folder contains test files for email attachment parsing.

## Files

1. **test-document.txt** - Plain text file with UTF-8 encoding
2. **test-image.svg** - SVG vector image
3. **test-data.csv** - CSV data file
4. **test-html.html** - HTML file

## Usage

Use `send-test-email-with-attachments.php` to send an email with these attachments.
